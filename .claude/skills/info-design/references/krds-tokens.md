### 폰트 패밀리

- **`--font-sans`** = `'Pretendard GOV', 'Malgun Gothic', 'apple sd gothic neo', sans-serif`
- 직접 `font-family: ...` 작성 금지

### 1rem = 10px 트릭 (KRDS 명시 채택)

- `html { font-size: 62.5% }` 적용 → 1rem = 10px
- 모든 KRDS 토큰이 0.1rem 단위로 정의됨 (예: 1.6rem = 16px)
- `reset.css`에서 자동 처리. 제거하면 시스템 깨짐
